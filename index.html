Projects

Deft Debate

Java web app that allows users to register, debate, and vote! Users can search debates by category, resolution, status, date, or username. Users have a dashboard and profile. Moderators can edit, delete, publish, or unpublish debates as well as create other moderators. Functionality is locked using Spring security, Hibernate, and BCrypt.

Built using Test-Driven Development, REST, Spring, MVC, JDBC, MySQL, Ajax, JSTL, Javascript, jQuery, HTML/CSS and Bootstrap.

Blog Builder Website

A collaborative, dynamic blog website that gives the admin the ability to create blogs and static pages. Admin functionality varies based on login credentials.

Made using Java, JSP, Spring Core, Spring MVC, Spring Security, MySQL, JDBC, Ajax, JSON, jQuery, Javascript, JSTL, HTML, Bootstrap & Git.

Vending Machine Web App

Digital vending machine that allows users to insert money, choose items, and returns change. Includes an admin page for CRUDing items that appears only after logging in.

Built with Java, JSP, Spring Core, MVC, and Security, MySQL, JDBC, TDD, RESTful, Javascript, Ajax, Json.

Flooring Machine Console App

Java CRUD console app to create and manage flooring orders for a flooring business. The orders are written to files by date.

The app was made using Java, Spring MVC, dependency injection, validation, and file I/O.
